# デイトレード エントリー/エグジット ビジュアライザー

証券会社の約定履歴から、その日のエントリーポイント・イグジットポイント・損益を自動計算し、チャート上に可視化するツールです。単一のHTMLファイルとして動作し、サーバーやビルド環境を必要としません。ブラウザで開くだけで使えます。

## デモ / 使い方

`daytrade_visualizer.html` をブラウザで直接開いてください。

1. 「データ入力」欄に約定履歴を貼り付ける（またはCSVファイルを選択）
2. 「グラフを更新」を押す
3. 必要に応じて「実際の値動きデータ」を重ねる、シンボル名を入力する、CSV/HTMLでエクスポートする

## 主な機能

### 1. チャート可視化
- 約定価格の推移（グレーの破線）
- 各トレードのエントリー（丸）→イグジット（ひし形）を結ぶ円弧（ロングは上向き、ショートは下向き）
- 利益／損失で色分け（利益＝赤、損失＝青。日本の株価表示に準拠）
- 昼休み（11:30〜12:30）のシェーディング
- 累積損益カーブ
- ホバーで詳細ツールチップ表示

### 2. サマリー統計
- 当日損益、トレード数、勝率（勝敗比率バー付き）
- プロフィットファクター
- 平均利益／平均損失、最大利益／最大損失（それぞれ独立したカードで正しい色分け）

### 3. 対応データ形式（自動判定）
「データ入力」欄に貼り付けた内容から自動的に形式を判定します。

| 形式 | 説明 |
|---|---|
| SBI証券 日計り信用成績表 | タブ区切り。列：約定日・約定成立時刻・受渡方法（新規/決済）・売買区分（買い/売り）・単価・数量・約定金額 |
| 楽天証券 信用取引 約定履歴 | ダブルクォート区切り（カンマ・タブ・改行いずれも可）。列：約定日（日付＋時刻）・コード・銘柄名・取引（信用新規/信用返済）・売買（買建/売建/買埋/売埋）・約定数量・約定単価 など |
| 本ツールのエクスポートCSV | 「トレード結果をCSVでエクスポート」で出力した形式をそのまま再インポート可能 |

**楽天証券フォーマットの注意点**
- 約定履歴に複数日・複数銘柄が混在している場合、約定件数が最も多い「日付＋銘柄コード」の組み合わせを自動抽出し、対象外にしたデータは画面に注記されます。
- 時刻情報（HH:MM）が含まれていない行は、正確な時系列表示ができないため除外されます。

### 4. 数量（分割決済・合算決済）への対応
新規建てと決済の数量が一致しない場合（例：200株建てて100株ずつ2回に分けて決済）も、FIFO（先入れ先出し）でロットを自動的に分割・合算してトレードを計算します。

### 5. 実際の値動きデータの重ね合わせ
自分の約定価格だけでなく、銘柄の実際の値動きをチャートに重ねられます。

| 入力形式 | 列 | 表示 |
|---|---|---|
| 歩み値 | 時刻, 価格[, 出来高] | シアン色の実線 |
| 分足OHLC | 時刻, 始値, 高値, 安値, 終値 | 簡易ローソク足（赤＝陽線、青＝陰線） |
| TradingView CSVエクスポート | time, open, high, low, close, Volume（ISO日時 or UNIXタイム、複数日分でも対象日を自動抽出） | 簡易ローソク足 |

CSVファイルの直接アップロードにも対応（UTF-8／Shift-JISを自動判定）。

### 6. エクスポート機能
- **トレード結果CSV**：確定したトレード（方向・時刻・価格・数量・損益）をCSVでダウンロード
- **分析用HTMLスナップショット**：チャート（SVG）・サマリー・トレード一覧に加え、末尾に構造化データ（JSON）を埋め込んだ単体HTMLファイルを出力。生成AIにそのまま渡して分析を依頼できます。ファイル名には日付と銘柄名が自動的に含まれます。

## 技術情報

- 単一のHTMLファイル（`daytrade_visualizer.html`）に HTML / CSS / JavaScript を内包
- 外部依存はGoogle Fonts（Web フォント）のみ。ライブラリやビルドツールは使用していません
- チャートは全て素のSVGで描画（Canvas / 外部チャートライブラリ不使用）
- ブラウザの `localStorage` / `sessionStorage` は使用していません（すべてメモリ上で完結）

## 制限事項

- **1画面につき1銘柄が前提**です。複数銘柄のデータを貼り付けた場合は、最も件数の多い銘柄のみが自動抽出されます。
- FIFOマッチングは「新規→決済」が同一の抽出データ内で完結していることを前提としています。日をまたいで建てたポジションを決済している場合、対応する新規建てが見つからず警告が表示されます（計算自体は継続されます）。
- 外部API（例：J-Quants等）への直接接続は行っていません。ブラウザのサンドボックス上の制約により、任意の外部APIへの直接fetchは想定通りに動作しない場合があります。値動きデータは手元のCSV/歩み値を貼り付ける方式を採用しています。
- 時刻情報のない約定データ（日付のみ）は、正確な時系列チャートを描けないため対象外としています。
